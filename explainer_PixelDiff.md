## Problem Description

Visual regressions often slip through code reviews and manual QA, reaching end users before teams catch them. PixelDiff solves this by providing a pixel-level image comparison tool that highlights exact differences between two images, draws bounding boxes around changed regions, and calculates a similarity score. It gives developers and designers a fast, intuitive way to detect unintended visual changes before they ship.
